# Test sessions

Completed feedback sessions for goodenough, one Markdown file per session.

Naming: `YYYY-MM-DD-yourtag.md` (e.g. `2026-05-14-oriol-mex-batteries.md`).

See [../FEEDBACK.md](../FEEDBACK.md) for the rubric, the per-session template, and the worked SHAP example.

Submit by opening a PR. If you're a collaborator on `nzipl-claude-kit`, self-merge. Otherwise Gilberto merges sessions during weekly triage (Mondays).

**Never opened a PR before?** [../HOW_TO_SUBMIT.md](../HOW_TO_SUBMIT.md) walks through it step by step, browser-only path included.
